# Documentation iDryer pour développeurs

Référence technique pour les développeurs construisant des appareils, des intégrations et des services compatibles avec l'écosystème iDryer.

## Sections

- **[Core](core/)** — Bibliothèque ESP32 pour les appareils compatibles iDryer. Gère le provisionnement WiFi, l'enregistrement cloud, la session TLS MQTT, le routage des commandes et la télémétrie. Vous n'écrivez que la logique spécifique à l'appareil.
- **[Construis ton propre appareil](byod/)** — Fondamentaux d'électronique et de firmware pour les makers : contrôleurs, capteurs, chauffages, circuits de puissance, interfaces et guides pratiques.

## Documentation utilisateur

Les guides d'assemblage, de flashage et d'utilisation se trouvent sur le site principal du projet : [docs.idryer.org](https://docs.idryer.org/).
