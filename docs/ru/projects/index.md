# Экосистема iDryer

![iDryer Ecosystem](../../img/iDryer_all.png)

## О проекте

iDryer — это **открытая экосистема** для сушки филамента и повышения качества 3D-печати. Проект включает аппаратное обеспечение, прошивку, облачный портал и мобильное приложение, созданные сообществом энтузиастов 3D-печати.

Вся документация, исходный код и чертежи доступны открыто. Вы можете собрать собственное устройство, модифицировать прошивку, интегрировать с вашей системой или способствовать развитию проекта.

## Основные продукты

### iDryer
Устройство для хранения и сушки филамента с точным контролем влажности и температуры. Поддерживает множество типов пластиков (PLA, PETG, ABS, Nylon и другие).

- **Документация**: сборка, прошивка, запуск
- **Версии**: от простых настольных сушилок до встроенных в печати
- **Сообщество**: примеры использования, мод-паки, интеграции

### iHeater
Модульная система обогрева камер для 3D-принтеров. Обеспечивает стабильную температуру для высокотемпературных пластиков.

### iDryer Storage
Сетевое хранилище с поддержкой облачной синхронизации и API для интеграции с кастомными системами управления.

### iDryer Portal
Облачный портал и мобильное приложение для мониторинга и управления устройствами из любой точки мира. Отслеживание статистики использования, уведомления о критических событиях.
