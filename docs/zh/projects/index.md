# iDryer生态系统

![iDryer生态系统](../../img/iDryer_all.png)

## 关于项目

iDryer是一个**开放生态系统**，用于灯丝干燥和提高3D打印质量。该项目包括硬件、固件、云门户和移动应用程序，为3D打印爱好者创建，并通过他们的想法、讨论和需求而不断发展。

所有文档、源代码和原理图均可公开获取。您可以构建自己的设备、修改固件、将其与系统集成、分享经验并为项目的发展做出贡献。

## 主要产品

### iDryer
一种具有精确湿度和温度控制的灯丝存储和干燥装置。支持所有主要塑料类型——从PLA和PETG到高温材料（ABS、Nylon、聚碳酸酯等）。干燥温度完全可调整，取决于您的灯丝类型和目标——从温和模式到110°C及以上。设计是模块化的：您可以根据需求和预算构建设备，从简单的桌面干燥器到整个打印机队的集成管理系统。

- **组装**: 详细说明、零件清单、原理图
- **固件**: 源代码、针对您的设备的定制
- **集成**: Klipper、Home Assistant、定制解决方案

### iHeater
用于3D打印机喷头室的模块化加热系统。为高温材料（Nylon、PEEK、聚碳酸酯等）提供稳定温度。与Bambu Lab、Klipper和其他平台集成。通过iDryer门户管理——实时监控温度、接收警报、为每种材料配置配置文件。

### iDryer Storage
灯丝存储管理系统。允许您在不需要记住架子地址的情况下完美地保存库存——只需将线圈放在其位置，系统会告诉您放在哪里。门户存储位置、状态、打开日期和使用历史记录以及实际重量。

### iDryer Portal & App
用于监控和管理生态系统中所有设备的统一平台。
**主要功能:**

- **灯丝知识库**: 与社区论坛集成——材料评论、用户体验、推荐在一个地方
- **预设**: 为每根灯丝共享干燥和打印配置文件、查看最有效的方法
- **干燥动态**: 实时监控水分流失、跟踪干燥参数与打印质量之间的关联
- **设备验证**: 每个干燥器都获得唯一标识符、确保参数合规性和构建质量
- **结果共享**: 发布您的干燥会话结果以及打印结果——真正有效的东西的客观图景

门户和移动应用使用单个数据库工作并实时同步。
